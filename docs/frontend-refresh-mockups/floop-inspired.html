Implementation notes

The useful parts can be adopted without overcommitting to a literal visual theme.

Apply to code

Keep

  • Lane language for work routing and parallel agent ownership.
  • Depth language for summary, evidence, and raw execution logs.
  • Deck language for exceptions requiring operator judgment.
  • A calmer aquatic palette with off-white tile, deep green, and restrained aqua.

Avoid

  • Novelty copy that makes operational work feel unserious.
  • Animated water effects behind dense text.
  • Too many blues or a one-note aquatic palette.
  • Replacing precise workflow language with vague metaphor labels.
