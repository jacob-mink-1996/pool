Static artboards for alignment before changing the React app.

Eight concrete directions for making Floop feel intentional.

These mockups keep the existing product model: project rail, board, ticket detail, evidence, merge readiness, activity, and artifacts. The differences are information hierarchy, density, color discipline, and the primary operating metaphor.

Visual thesisCalm control room, matte surfaces, precise state, and evidence-forward workflow.
Content planOrient the workspace, move tickets, inspect evidence, decide merge or rework.
Motion thesisPanel reveals, command focus, and state-transition movement rather than decorative animation.
